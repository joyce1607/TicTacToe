# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project allows two players to play the classic 3×3 grid game in a modern, responsive interface.

## ✨ Features

* Two-player gameplay (X vs O)
* Automatic winner detection
* Draw detection
* Turn indicator
* Restart game functionality
* Responsive and user-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```
index.html
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start playing the game.
